Slider je kreiran uz pomoc slick slidera i editovanja slick.js filea, kako bi prikazao 2 rowa, uvjetovano if row>0.
Nazalost nisam u potpunosti replicirao izgled.
Primjetio sam da sam malo zahrdao u editovanju jer sam zadnjih 2-3 mjeseca ucio prvo javaScript i nakon toga Angular.
Medutim, sad cu ponovo izvjezbati napredni CSS i HTML5, tako da ce ovo biti ok.
